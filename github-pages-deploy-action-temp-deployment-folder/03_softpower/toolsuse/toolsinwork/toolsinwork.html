# 工作中使用的工具探讨

每个行业都有其工具，软件开发也是如此。 优秀的程序员比其他人更了解自己的工具，并且比普通程序员更有权使用更强大和先进的工具。

本文章主要对日常工作中所使用工具的某些方便点进行沟通共享，所涉及的工具不一定是软件开发方面最有效率、最主流的。
