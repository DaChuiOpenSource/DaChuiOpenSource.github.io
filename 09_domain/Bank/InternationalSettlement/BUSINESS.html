# 跨境金融

# 1. 外汇账户类

# 1.1. 外债账户

银行为办理过外债签约登记的企业和非银行金融机构开立外债专用账户，用于接收债权人汇入的外债资金。

# 1.2. 外汇资本金账户

外商投资企业外汇资本金专户用于存放境外投资者汇入外商投资企业的外币出资部分，以境内主体名义开立。

# 1.3. 外汇结算账户

进出口企业、提供跨境服务的服务型企业等办理经常项目外汇收支开立的账户。

# 2. 国际结算类

# 2.1. 汇入汇款

我行根据国外汇出行指示将款项解付给指定收款人的业务，用于满足国际间资金汇划结算需求。

1.我行收到国外汇出行发出的汇款指示电报；
2.我行对上述指示电报审核无误并收妥头寸后， 向收款人解付汇款。

通过待处理跨境汇入交易查询，查出国结端待办理的入账业务

# 2.2. 汇出汇款

我行接受汇款人的委托，以约定汇款方式委托代理行将一定金额的款项付给指定收款人的业务，用于满足国际间资金汇划结算需求。

1.汇款人通过线下或者线上渠道向我行提交《汇出汇款申请书》；
2.我行经审核后向代理行发出汇款指示电报；
3.代理行按我行指示向收款人解付汇款。

# 2.3. 进口信用证

我行依照开证申请人（进口商）的要求和指示，承诺在符合信用证条款情况下，凭规定的单据，向第三者（受益人）或其指定人进行承付，或授权另一银行进行承付或议付。

1.进口商提交开证申请书等资料，我行经审核后凭保证金或占用授信额度开出信用证；
2.出口商收到信用证通知后，按信用证条款备货装运；
3.出口商提交信用证规定单据，交单行将单据寄往我行要求付款或承付；
4.我行收到单据后，经审核无误向出口商付款或承付后将单据交付进口商；
5.承付到期日，进口商通过我行向出口商付款。

# 2.4. 出口信用证

我行根据国外银行开来的信用证及客户（出口商）交来的单据，为出口商办理审单、寄单、索汇、收汇的一种结算业务。

1.我行收到国外开来的信用证，证实其真实性后通知受益人（出口商）；
2.受益人（出口商）按信用证条款备货装运后，向我行提交出口单据；
3.我行审核单据后，将单据寄往国外开证行或指定行进行索汇；
4.国外开证行收到单据后提示给信用证项下开证申请人；
5.国外开证行到期向我行付款，我行向受益人（出口商）解付。

我行受出口商委托，凭其提交的出口商业单据和金融票据通过国外代收行向进口商收取款项。

# 2.5. 跟单托收

我行受出口商委托，凭其提交的出口商业单据和金融票据通过国外代收行向进口商收取款项。

1.境内出口商将全套出口单据及跟单托收项下交单申请书交至我行；
2.我行审单后缮制托收面函连同全套单据寄往代收行进行索汇；
3.代收行收到全套托收单据，并提示给境外进口商；
4.进口商付款/承兑赎单；
5.代收行将款项付至我行，我行向出口商解付。

# 2.6. 跟单代收

我行作为进口方银行（代收行）根据出口方银行（托收行）的委托，向进口方（付款人）提示单据，并根据托收行的指示办理付款交单或承兑交单手续。

1.国外出口商备货发运后，将有关单据提交所在银行办理托收；
2.出口商所在银行将托收单据寄至我行，我行按照指示通知进口商付款（D/P）或承兑（D/A）；
3.进口商通过我行向出口商付款或承兑后，我行将单据交付进口商；
4.D/A 项下承兑到期后，进口商通过我行向出口商付款。

# 2.7. 跨境人民币

我行应客户申请，为客户提供的跨境人民币结算相关的各类产品，包括结算、融资、担保、资金交易等产品，满足客户对跨境人民币结算业务的综合需求。目前我行现有的国际结算和贸易融资等产品均适用于跨境人民币结算业务。

客户办理跨境人民币业务时，需向我行提供中国人民银行规定的《跨境贸易人民币结算出口收款说明》和《跨境贸易人民币结算进口付款说明》，其他具体业务办理流程同我行国际结算、贸易融资等业务。

# 3. 国际贸易融资

# 3.1. 出口代付

出口代付业务是指我行作为委托行，委托银行同业作为代付行代为支付客户出口款项，我行承担于约定的日期向代付行偿还融资本金、利息并支付相关费用的责任。包括出口信用证项下代付、出口托收项下代付和出口 T/T 项下代付。

1.客户向我行申请综合授信额度，我行审查同意；
2.客户向我行申请叙作出口代付业务，并与我行签订业务协议；
3.我行对资料进行审核；
4.审核通过后，通知代付行按照我行指令付款给出口商；
5.融资到期日，客户按协议约定偿还我行融资本息及相关费用，我行向代付行偿付代付本息。

# 3.2. 出口发票融资

在 T/T 汇款项下，申请人（出口商）将现在或将来的基于与进口商订立的出口销售合同项下产生的应收账款质押给我行，由我行为其提供贸易融资服务。

1.客户向我行申请综合授信额度，我行审查同意与申请人签订协议，并落实有关担保；
2.申请人向我行提交出口发票、出口订单等相关资料；
3.我行审核通过后，发放融资款项；
4.融资到期时，客户归还我行融资本息。

# 3.3. 出口订单融资

为支持贸易项下卖方备货出运，应卖方的申请根据其提交的贸易合同或采购订单而向其提供的用于订单项下货物采购、生产和装运的专项贸易融资。

已取得国外订单，在生产备货阶段有融资需求的出口企业

1.客户向我行申请综合授信额度，我行审查同意与申请人签订协议，并落实有关担保；
2.申请人向我行提交出口订单、采购订单等相关资料；
3.我行审核通过后，发放融资款项并受托支付给供货商；
4.融资到期时，客户归还我行融资本息。

# 3.4. 出口信用贷

我行基于出口客户的历史收汇数据，为企业提供纯信用的贸易融资服务。

河南省内信用记录良好、经营情况稳定的中小出口企业

1.客户向我行申请授信额度，我行审查同意；
2.客户与我行签订业务协议；
3.我行审查通过资料后，向客户进行放款；
4.融资到期时，客户归还我行融资本息。

# 3.5. 进口代付

我行作为委托行，委托银行同业作为代付行代进口商对外付款，我行承担于约定的日期向代付行偿还融资本金、利息并支付相关费用的责任。包括进口信用证项下代付、进口代收项下代付和进口 T/T 项下代付。

出口商希望获得即期付款，但进口商希望远期付款并获得资金融通。

1.客户向我行申请综合授信额度，我行审查同意；
2.客户向我行申请叙作进口代付业务，并与我行签订业务协议；
3.我行对资料进行审核；
4.审核通过后，通知代付行按照我行指令付款；
5.代付行收到指示后按照我行的付款路线履行付款责任；
6.融资到期日，客户按协议约定偿还我行融资本息及相关费用，我行向代付行偿付代付本息。

# 3.6. 进口 T/T 融资

我行应付款申请人的要求，在审核其汇款业务依法合规，并落实合法有效的担保后，为其办理融资并代为对外付款的业务。

进口商遇到临时资金周转困难，无法按时付款；

1.客户向我行申请综合授信额度，我行审查同意；
2.进口商向我行提交汇出汇款融资申请书以及相关协议，我行落实有关的担保后代进口商对外垫付融资款项；
3.进口商到期向我行付款，用以归还融资款项。

# 3.7. 进口押汇

我行应客户（开证申请人或代收付款人）要求，与其达成信用证或进口代收项下单据及货物所有权归我行所有的协议后，我行以信托收据的方式向其释放单据并先行对外付款的行为。

进口商遇到临时资金周转困难，无法按时付款赎单；

1.客户向我行申请综合授信额度，我行审查同意；
2.进口商向我行提供进口押汇申请书以及相关协议，我行落实有关担保后代进口商对外垫付押汇款项，并将单据交付进口商；
4.进口商到期向我行付款，用以归还押汇款项。

# 4. 国内贸易融资

# 4.1. 议付

信用证的议付是议付行在受益人交单相符的情况下，用自有资金先于开证行向受益人垫付价款。但是，如果日后开证行没有对议付行支付该信用证项下的价款，那么，议付行有对受益人追索之前垫付的价款的权利。

1.卖方与我行签订融资协议，向我行提交出口单据；
2.我行经审核单据后，将单据寄往开证行进行索汇；
3.开证行收到单据后向我行进行付款确认；
4.我行在收到开证行付款确认后，客户应向我行提交国内信用证议付合同等协议，我行将议付款项入卖方账户；
5.开证行到期向我行付款，我行用以归还议付款项。

# 4.2. 福费廷

福费廷业务又称单据包买，是指包买商无追索权地购买已由开证行承兑或承诺付款的信用证项下应收账款的贸易融资业务。 福费廷是银行买断受益人的未来收益权，继续做福费廷的银行或机构，在对受益人支付对价之后，取得了该信用证项下的收益权。但是，如果开证行日后不付款，继续做福费廷的银行或机构没有向受益人追索的权利。

1.客户向我行提交业务申请并签订相关业务协议；
2.我行审批通过后，发放融资款项。

# 4.3. 国内信用证

国内信用证指的是，我行按照申请人的请求向国内受益人开立的，凭符合信用证规定的单据向国内受益人付款的一种结算方式。

1.买卖双方签订贸易合同，约定可通过国内信用证结算；
2.买方作为申请人向开证行申请开证，全额保证金或部分占用授信额度；
3.开证行开出信用证，发送到通知行；
4.通知行将信用证通知给受益人（卖方），受益人按要求发货并提交相关单据；
5.开证行确认单证一致后向卖方付款。

# 5. 跨境担保类

# 5.1. 非融资性保函

我行作为担保人，应被担保人或反担保人申请，以保函或备用信用证等书面形式向受益人承诺，当被担保人未按照约定履行非融资性交易项下义务时，由我行代其履行赔偿或付款责任的行为。非融资性跨境担保主要类型包括：投标担保、履约担保、预付款担保、质量及维修担保、留置金担保、海事担保、关税担保、付款担保、延期付款担保、以实物偿还的补偿贸易以及其他的非融资性担保。

1.客户向我行申请综合授信额度，我行审查同意；
2.申请人向我行申请开立保函，签订相关业务协议，存入保证金或者占用授信额度；
3.我行审核通过，开出保函。

# 5.2. 融资性保函

我行作为担保人，应被担保人或反担保人申请，以保函或备用信用证等书面形式向受益人承诺，当被担保人未按照约定履行融资性交易项下义务时，由我行代其履行赔偿或付款责任的行为。融资性交易包括：借款、发行有价证券（不包括股票）、透支、融资租赁、以现汇偿还的补偿交易以及其他融资交易等。

1.客户向我行申请综合授信额度，我行审查同意；
2.申请人向我行申请开立保函，签订相关业务协议，存入保证金或者占用授信额度；
3.我行审核通过，开出保函。

# 6. 外汇资金类

# 6.1. 合作远期结售汇

经客户申请，由我行与具备经营远期结售汇业务资格的银行及其分支机构合作为客户办理远期结售汇相关业务。客户与我行及具备资格银行约定将来办理结汇或售汇的外币币种、金额、汇率和期限；到期外汇收入或支出发生时，即按照约定的币种、金额、期限、汇率办理结汇或售汇。

境内机构以及经国家外汇管理局批准的其他客户。

1.客户与我行、合作银行签订三方合作协议；
2.客户向我行询价并接受我行报价，提交远期结售汇业务授权委托书等相关资料；
3.我行收到客户提交的远期结售汇业务申请后，按照国家外汇管理局及我行有关结售汇业务规定，对客户的业务申请进行审核；
4.客户交纳保证金或者占用客户授信额度；
5.我行向具备资格银行提出远期结售汇交易申请。
6.到期交割。

# 6.2. 即期结售汇

即期结售汇是指交易双方以即时汇率成交并最迟在成交后的第二个营业日进行资金交割的交易。
结汇业务：是指客户将外汇卖给我行，我行根据交易行为发生时的汇率支付等值人民币的行为。
售汇（购汇）业务：是指我行将外汇卖给客户，并根据交易行为发生时的汇率收取等值人民币的行为。 购汇时，账户、牌价、国结提交核心记账

境内机构、个人以及经国家外汇管理局批准的其他客户。

客户根据真实换汇需要，通过我行线下以及线上渠道办理即期结售汇交易，客户办理即期结售汇业务应符合有关国家外汇管理局结售汇业务管理规定。
