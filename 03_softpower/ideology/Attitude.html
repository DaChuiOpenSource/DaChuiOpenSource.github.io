# 消极与积极思考

# 1. 何为消极

所谓消极，是指在精神上困于一隅，只能一味片面思考的状态。

# 2. 何为积极

所谓积极，是指能从多角度看待事物，思路丰富而不局限的状态。在这种状态下，人的眼界很开阔，所以能用正面心态去看待下属。

# 3. 积极思考

人在无意识的状态下，往往只能看到片面的一方面，比如看到对方的缺点，忽视对方的优点。为此，多角度的思考——积极思考——是不可或缺的。消极思考并不是好事，希望大家都能积极思考。

积极思考是顺畅交流的基础。
避免自己进去无意识状态。
