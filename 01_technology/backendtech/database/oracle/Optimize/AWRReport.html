# Oracle AWR Report

# 1. AWR介绍

对于局部SQL调优，我们可以直接使用执行计划等调优，而对于整个系统可以用Oracle系统自带的报告对系统进行整体分析。

AWR全称Automatic Workload Repository，自动负载信息库，是Oracle 10g版本后推出的一种性能收集和分析工具，提供了一个时间段内整个系统的报表数据。通过AWR报告，可以分析指定的时间段内数据库系统的性能。
