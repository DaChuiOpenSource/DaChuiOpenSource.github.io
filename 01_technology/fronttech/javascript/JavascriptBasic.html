string: It is the input value, which contains an integer and is to be converted. radix: Specify which number system is being used in terms of computer, For example, 2-Binary, 8=octal, etc., till 36. By default, the numbers are converted into decimal values. The parseInt() is a built-in JavaScript that parses a string argument and returns an integer of the specified radix. If the string contains any alphabet, it stops parsing at that moment or parses only integer before the alphabets.

# 3.1. $(function() {})

$(function() {}) 是$(document).ready(function()的简写 是在DOM加载完成后执行的回调函数，并且只会执行一次。

DOM 在第一次页面加载完毕后，就在内存里了，无论后面怎么通过ajax的方式去局部修改html页面，都只是对内存中的 DOM 树进行修改，而 DOM在第一次页面加载完毕后就已经加载完毕了。后面 js文件（动态加载或者 head 中加载）再使用到 $(function() {})函数肯定会执行的。 s
