# 3. 调试项目

  1. VSCode打开插件项目
  2. 在源码需要调试的地方增加断点
  3. F5自动打开新的VSCode进行调试

# 4. 发布项目

安装vsce插件管理工具
npm install -g vsce
打包
vsce package

登录vscs
vsce login DaChuiOpenSource
预发布
yarn run vscode:prepublish
发布
vsce publish
