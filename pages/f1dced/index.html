# ElectronJS

# GetStart

创建项目目录并初始化工程
mkdir DCMarkdown && cd DCMarkdown
npm init

安装electron依赖
//npm install --save-dev electron
cnpm install --save-dev electron

注意 由于国内网络访问原因,使用cnpm进行安装

添加启动命令

  "scripts": {
    "start": "electron ."
  }

将页面装载浏览窗口

const { app, BrowserWindow } = require('electron')

const createWindow = () => {
    const win = new BrowserWindow({
        width: 800,
        height: 600
    })
    win.loadFile('index.html')
}

app.whenReady().then(() => {
    createWindow()
})

打包、发布应用
cnpm install --save-dev @electron-forge/cli
npx electron-forge import // set electron-forge 设置打包的package.json配置及依赖

npm run package --打包
npm run make --发布

打包慢可以添加setenv脚本

"scripts": {
    "test": "test",
    "start": "electron-forge start",
    "setenv": "set ELECTRON_MIRROR=http://npm.taobao.org/mirrors/electron/",
    "package": "npm run setenv && electron-forge package",
    "make": "npm run setenv && electron-forge make"
}

# monitorAndRestart

加入nodemon,它可以监控node.js 源代码的变化,并自动重启应用
npm i nodemon --D
or
yarn add nodemon --dev
scripts："start": "nodemon --watch index.js --exec electron ."

# Others

查看镜像地址
npm config ls -l | grep registry
registry = "https://registry.npmjs.org/"

npm config set registry http://registry.npm.taobao.org/
npm config set ELECTRON_MIRROR http://npm.taobao.org/mirrors/electron/

npm config ls -l | grep ELECTRON_MIRROR
