# 1.2. 配置文件介绍

  1. config.js
    位置：docs/.vuepress/config.js
    功能：Entry file of configuration。应用的配置文件，包括标题、描述、导航、侧边栏及主体配置。

# 1.3. 页面内置

$description config.js文件中描述
$frontmatter 每个页面的frontmatter数据
$page 当前页面的信息, 包括frontmatter、path、title等数据
$site 当前站点信息, 包括title、description、base、headTags、pages、themeConfig等数据
$siteTitle 站点标题
$themeConfig 主题配置
$title 标题
